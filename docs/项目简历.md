# MimoLin Space — 个人作品展示与内容管理平台

**独立开发 · 已上线 mimolin.space**

**技术栈：**Java 17、Spring Boot 3、Spring Security、JPA、PostgreSQL、Flyway；Docker、Nginx、React、TypeScript

**项目描述：**个人作品展示与内容管理网站，前台涵盖项目展示、技术博客、摄影图册、访客留言及个人介绍等功能；后台统一管理项目、博客、图册、评论、个人资料、简历版本和首页相册，支持内容新增、编辑、删除及草稿、发布、隐藏等状态流转。实现图片 EXIF 自动校正、多分辨率图片生成、评论审核和简历 PDF 多版本管理，并完成线上部署。

**项目亮点：**

1. **后端架构与接口设计：**采用 Controller→Service→Repository 分层，业务逻辑与事务边界集中于 Service 层；使用 Java Record DTO 隔离持久化实体，结合 Bean Validation 完成参数校验与模型转换；通过 @RestControllerAdvice 统一处理业务、认证及校验异常。设计 50 个 RESTful API，覆盖 6 个核心业务域。
2. **认证与访问控制：**基于 Spring Security 构建会话认证，使用 AuthenticationManager 校验凭证、BCrypt 存储密码，并将安全上下文写入 HttpSession。采用 CookieCsrfTokenRepository 校验 SPA 写请求；访客接口公开，/api/admin/**及密码修改需认证，未登录返回 401，启动时自动初始化管理员账户。
3. **数据建模与版本迁移：**基于 PostgreSQL 设计 9 张核心业务表，通过 CHECK 约束角色、内容状态及评论关联类型，结合 CASCADE/SET NULL 外键策略管理数据生命周期，并以部分唯一索引保证仅一个简历版本处于激活状态。使用 Flyway 管理 13 次 Schema 迁移，并针对状态筛选、排序与分页查询建立复合索引。
4. **内容状态与媒体处理：**统一以 ContentStatus 管理项目、博客与图册的草稿、发布、隐藏状态，并校验父子内容可见性。图片上传后通过自研 JPEG EXIF 解析器完成方向矫正，超限图片按最长边 1600px 压缩；图册额外生成 640px 缩略图与 1600px 中图，按场景加载以降低传输体积。
5. **测试与生产部署：**基于 JUnit 5、Spring Boot Test 与 MockMvc 编写接口集成测试，覆盖认证、内容 CRUD、评论审核、图片处理及可见性过滤，并通过单元测试验证 EXIF 矫正与衍生图生成。生产环境由 systemd 托管后端，Nginx 提供静态资源服务与 API 反向代理，PostgreSQL 通过 Docker Compose 部署。
