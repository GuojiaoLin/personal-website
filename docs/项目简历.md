# 个人作品集网站 —— 全栈 CMS 博客与内容管理平台

**独立开发 · 已上线 mimolin.space**&emsp;&emsp;&emsp;&emsp;**2025.XX – 至今**

**技术栈：** React 18、TypeScript、Vite、Tailwind CSS、Framer Motion、Radix UI、Spring Boot 3、Spring Security、Spring Data JPA、PostgreSQL、Flyway、Docker、Nginx、EdgeOne CDN、Sharp / ImageIO

**项目描述：** 独立设计并开发的前后端分离全栈个人网站。前台 7 个页面（首页、项目经历、技术博客、摄影图册、访客留言、关于我、简历），后台覆盖项目、博客、图册、评论、简历版本、个人页 6 大模块的完整 CRUD 管理。从设计、开发、部署到运维一人完成，3 周落地 9 张数据表、40+ REST 接口、13 个 Flyway 迁移版本，已上线运行于腾讯云。

**项目亮点：**

- **手写 Markdown 渲染引擎：** 未依赖 react-markdown 等三方库，自行实现 Markdown → React 元素的完整转换（标题、列表、表格、代码块、链接、图片）；扩展 `![[filename|width]]` 自定义语法实现图片宽度控制与路径自动映射；集成 Mermaid 在博客内渲染流程图与架构图。
- **认证与分层授权：** BCrypt + HttpSession 实现管理员登录，`/api/admin/**` 统一鉴权拦截；启用 CSRF token，前端写操作自动携带 `X-XSRF-TOKEN`；站主回复自动跳过审核流。
- **内容三态管理与评论审核：** 博客、项目、图册统一实现 DRAFT → PUBLISHED → HIDDEN 状态机，公开接口仅返回已发布内容；评论支持博客评论与留言板双场景，树形回复 + 级联删除，访客留言默认待审核。
- **Flyway 版本化迁移：** 13 个迁移脚本管理 9 张核心表的 schema 演进（建表、复合索引、种子数据迁移、图册派生图字段）；简历版本表通过部分唯一索引在数据库层保证有且仅有一个激活版本。
- **响应式瀑布流图册：** CSS Grid + JS 列数计算实现 1/2/3 列自适应布局，配合 Lightbox 沉浸式大图预览；API 不可用时自动降级到本地图片目录，保证前端独立可用。
- **媒体资源与图片优化：** 图片通过外键或业务槽位隔离归属，后台支持博客配图、项目图、About 图和图册上传；大图上传后压缩、修正 EXIF 方向，图册额外生成 thumbnail / medium 派生图。
- **前端工程化与动效：** React.lazy + Suspense 按路由代码分割；Framer Motion 实现页面过渡、毛玻璃导航栏、卡片悬浮光晕；后台 3200+ 行单页覆盖 6 大模块 CRUD 与实时 Markdown 预览。
- **生产部署与 CDN 治理：** Nginx 静态托管 + API/uploads 反代，systemd 托管后端，Docker Compose 运行 PostgreSQL；上线后通过 curl 指纹对比定位 EdgeOne CDN 缓存旧页面问题，制定 index.html 不缓存 + 静态资源长缓存策略。
