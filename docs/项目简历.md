# 个人作品集网站 —— 全栈 CMS 博客与内容管理平台

**独立开发 · 已上线 mimolin.space**&emsp;&emsp;&emsp;&emsp;**2025.XX – 至今**

**技术栈：** React 18、TypeScript、Vite、Tailwind CSS、Framer Motion、Radix UI、Spring Boot 3、Spring Security、Spring Data JPA、PostgreSQL、Flyway、Docker、nginx、EdgeOne CDN

**项目描述：** 面向求职展示的全栈个人网站，独立设计并实现前后端分离的 CMS 内容管理系统：前台包含首页、项目经历、技术博客、摄影图册、访客留言、关于我共 7 个页面；后台提供项目、博客、图册、评论、简历版本、个人页的完整 CRUD 管理。系统从设计、开发、部署到运维由一人独立完成，覆盖 8 张数据表、30+ REST 接口、10 个 Flyway 迁移版本，已上线运行于腾讯云服务器。

**项目亮点：**

- **手写 Markdown 渲染引擎：** 未使用 react-markdown 等第三方库，自行实现完整的 Markdown → React 元素转换，支持标题、列表、表格、代码块、链接、图片等标准语法；扩展 `![[filename|width]]` 自定义语法实现博客图片宽度控制与路径自动映射；内嵌 Mermaid 图表渲染，支持在博客中展示流程图与架构图。
- **Spring Security 认证与分层授权：** 基于 BCrypt + HttpSession 实现管理员登录，`/api/admin/**` 统一鉴权拦截；前端通过接口检测登录态，响应式切换导航栏文案与功能入口（登录框 ↔ 后台入口）；站主回复自动跳过评论审核流。
- **内容状态机与评论审核：** 博客、项目、图册统一实现 DRAFT → PUBLISHED → HIDDEN 三态管理，公开接口仅返回已发布内容；评论支持博客评论与留言板双场景，树形回复结构与级联删除，访客留言默认待审核，后台支持按状态过滤与批量管理。
- **Flyway 版本化数据库迁移：** 10 个迁移脚本管理 8 张核心表的 schema 演进，涵盖建表、复合索引、种子数据迁移（硬编码内容迁入数据库）；简历版本表通过部分唯一索引在数据库层保证有且仅有一个激活版本。
- **响应式瀑布流图册：** CSS Grid + JS 列数计算实现 1/2/3 列自适应瀑布流布局，配合 Lightbox 沉浸式大图预览；优先从后台 API 加载数据，API 不可用时自动降级到本地图片目录，保证前端独立可用。
- **博客媒体资源管理：** 图片通过外键关联到具体博客文章实现资源隔离，后台支持上传、一键插入正文、设为封面；文件上传落盘后通过 Spring ResourceHandler + nginx 反代对外提供访问。
- **前端工程化与交互动效：** 全部页面组件使用 React.lazy + Suspense 按路由代码分割；Framer Motion 实现页面过渡动画、毛玻璃导航栏、卡片悬浮光晕效果；后台 3200+ 行单页覆盖 6 大模块 CRUD 与实时 Markdown 预览。
- **生产部署与 CDN 治理：** nginx 静态托管 + API/uploads 反向代理，systemd 管理后端进程，Docker Compose 运行 PostgreSQL；上线后通过 curl 指纹对比定位到 EdgeOne CDN 缓存旧页面的问题，制定了 index.html 不缓存 + 静态资源长缓存的策略。
